# P-P3-654375
En este proyecto realizaremos varios modelos de redes neuronales convulucionales, todo esto con el afan de mostrar numeros de entre el 0 al 9 a la camara, ya sea de nuestra laptop o nuestro celular y que el programa nos arroje su predicion o que nos diga cual numero es el que nos esta mostrando.

Como ya mencione, realizaremos varios modelos de los cuales elegiremos a uno para ser nuestro modelo **Ganador**, pero no descartermos los demas, pues mientras se este reproduciendo nos mostrara la probabilad de este mismo numero, como a su vez nos mostrara el resultado y la probabilidad de los otros 5 modelos que no ganaron para comparar.
